# MindHives ERPNext Tools

Open source automation scripts and workflow tools built on ERPNext/Frappe,
developed by the Mind Hives team.

## What This Is
A collection of practical ERPNext customizations, AI workflow integrations,
and automation scripts for digital agencies managing multi-company environments.

## Tools Included
- ERPNext workflow automation scripts
- AI-powered document processing helpers
- Multi-company contact management utilities
- Twilio SMS and calling integrations for ERPNext
- Nextiva call forwarding configuration templates

## Who This Is For
- Digital agencies running ERPNext
- Businesses managing multiple companies in one ERP system
- Teams integrating AI tools into their daily workflows
